Grid to Flex

Flexbox fallbacks for popular UI solutions that use CSS Grid.

Things to Note: CSS grid allows for specified styling using the vertical area as well as horizontal. Flexbox doesn't allow for this, so we'll be using fixed heights in these examples. The components would also take the height of the internal elements. The code here is fully editable. To swap between implementations, use the toggles. You can also play with these in Codepen (which may have better syntax highlighting!).
